# Post-mortems

Monthly, last Friday of the month. The Chair runs it.

## Process

1. **Aggregate.** Pull every shipped piece from the prior 4 weeks across channels (Buffer analytics, Meta insights, email metrics, outreach log).
2. **Score by what mattered.** Not impressions or vanity metrics — clicks to `colophon.contact`, look-pass purchases (via `/api/buyers`), concierge brief submissions, applicant signups.
3. **What worked.** Three posts/ads/emails/outreach moments that produced. Why.
4. **What didn't.** Three that didn't. Why.
5. **What to adjust.** Concrete changes to the next month's plan.
6. **Brief to CEO.** One page, in chat.

## File naming

`social/post-mortems/YYYY-MM.md` — e.g., `2026-05.md`.
