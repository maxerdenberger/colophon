# Weekly planning

The Chair convenes the board every Monday 9am PT (scheduled task — to be wired once the board is live).

## Process

1. **Inputs gathered (Friday–Monday).** Organic Social pulls last week's Buffer analytics. Paid Performance pulls Meta insights + CAC against `/api/buyers`. Lifecycle pulls email open/click. Partnerships logs outreach replies. All drop notes in `social/planning/YYYY-WW.md`.
2. **Slate proposed (Monday morning).** Chair synthesizes the four threads into a one-page proposal: this week's posts, ad tests, emails, outreach. Soft-launch tripwires flagged inline.
3. **CEO sees it.** Chair posts the proposal to chat. CEO reacts (greenlight, edits, kill).
4. **Specialists execute.** Each lead ships their piece. Buffer queue, ad spend, email send, outreach DMs. Updates appended to the week file as they go.
5. **Friday close.** Chair appends a one-line status to the week file.

## File naming

`social/planning/YYYY-WW.md` — e.g., `2026-22.md` for week of May 25, 2026 (ISO week number).
